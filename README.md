## ✨ Funcionalidades

- Informações sobre experiência, skills e contato

- Design responsivo para desktop e mobile

## 🛠️ Tecnologias

- React

- JavaScript / TypeScript (se aplicável)

- CSS / Tailwind / Styled Components (escolha o que usou)

```bash

cd seu-repo-aqui
```
